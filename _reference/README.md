# _reference — design reference only (NOT active code)

These are the **retired standalone Vite apps** (`highend` = Meridian, `lowcost` = GreenLeaf) from before
the [BRD](../BRD_LEDGE_SITE_FACTORY.md) consolidated everything onto the one Next.js engine.

They are **kept only as design/content reference** for rewriting `meridian` and `greenleaf` as *looks*
inside the engine (`apps/sitebuilder`). They are **not** part of the workspaces, are **not** built, and
must **not** be deployed. Do not add clients here.

- Use them to copy the visual design, layout, copy, and the shape of `content.ts` into the engine's look.
- Once both looks are rewritten and confirmed at visual parity, this folder can be deleted.
